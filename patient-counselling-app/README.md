## Installation
Install [Node.js](https://nodejs.org), [Ionic & Cordova](https://ionicframework.com/framework) on your system.
Clone the repository and run

````
$ git clone "URL from github"
````
````
$ npm install
````

Then , add the platform for android/ios, as per your choice of device
````
$ ionic cordova platform add "PLATFORM_OF_CHOICE"
````

Make  new version of your app
````
$ ionic corodva build "PLATFORM_OF_CHOICE"
````
